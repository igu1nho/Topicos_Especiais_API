# Exercício Aula 6 - API

## Para executar o código

1. Instalar dependências
```bash
cd aula06
npm install
```

2. Inserir link de conexão do MongoDB
```bash
cd src
cd infrastructure
```
- Abrir arquivo **database.js**
- Substituir **link** pelo link de conexão do seu MongoDB


3. Rodar o projeto
```bash
npm start
```
